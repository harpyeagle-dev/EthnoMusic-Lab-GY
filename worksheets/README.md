## Worksheet Usage Guidelines

### For Students
- Worksheets build on each other—complete them in order
- There are no "wrong" answers for interpretive questions
- Use additional paper if you need more space
- Save all worksheets for your final portfolio

### For Instructors
- Worksheets can be completed individually or in small groups
- Allow flexibility in time allocations based on student needs
- Use worksheets as formative assessment tools
- Provide feedback that encourages deeper thinking

## Digital and Print Options

All worksheets are available as:
- **Markdown files** - for digital completion or conversion
- **Interactive forms** - for online submission *(coming soon)*

## Accessibility

Worksheets are designed to be accessible:
- Clear formatting and structure
- Space for extended responses
- Multiple question types (open-ended, checklist, table)
- Adaptable for different learning needs
