# Resources Directory

This directory contains supporting materials for the Baboon Dance Digital Curriculum.

## Directory Structure

```
resources/
├── audio/                  # Audio files and samples
├── images/                 # Visualizations, graphs, photos
├── guides/                 # Reference guides and tutorials
├── templates/              # Project templates and starter files
├── tutorials/              # Step-by-step technical tutorials
└── cultural-context.md     # Cultural background information
```

## Audio Files

**Location:** `audio/`

### Baboon Dance Recordings
- Sample performances from different regions
- Various durations (clips and full performances)
- High-quality WAV files for analysis
- Compressed MP3 versions for quick access

### Comparative Materials
- Festival of the Sun (Inti Raymi) recordings
- Other Indigenous music traditions for extended study

**Note:** Audio files should be obtained ethically with proper permissions and consent from knowledge holders and communities.

## Usage Guidelines

### For Students
- Resources are designed to support your learning
- Always cite sources properly in your work
- Respect cultural protocols around sensitive materials
- Ask if you're unsure about appropriate use

### For Instructors
- Adapt resources to your teaching context
- Ensure proper permissions before sharing outside classroom
- Update materials as new resources become available
- Share your own resources with the community

### For Community Members
- These resources aim to honor and preserve cultural heritage
- Feedback and corrections are welcomed and valued
- Community voices should guide how materials are used
- Materials can be restricted or removed at community request

## Obtaining Materials

Some resources may need to be:
- **Licensed or purchased** - Commercial audio or software
- **Requested with permission** - Cultural materials from communities
- **Created locally** - Generated through your own analysis work
- **Downloaded** - Open-access datasets and tools

## Copyright and Permissions

All materials should be used in accordance with:
- Copyright law and fair use provisions
- Traditional Knowledge (TK) protocols
- Informed consent agreements
- Attribution requirements
- Educational use limitations

## Contributing Resources

If you create valuable resources while using this curriculum:
- Consider sharing them with the community
- Ensure you have rights to share
- Provide clear documentation
- Credit collaborators and sources

## Support and Questions

For questions about resources:
- Check module-specific resource lists first
- Consult instructor or curriculum coordinator
- Reach out to cultural community liaisons
- Review ethical guidelines before proceeding

---

**Important:** Cultural materials should always be treated with respect and used ethically. When in doubt, consult with community members and follow their guidance.
